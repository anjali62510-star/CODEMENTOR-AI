## 📖 Overview

CodeMentor AI is an AI-powered career navigation platform designed to help students and aspiring developers build successful careers in technology.

Using an immersive ocean-inspired experience, users can track their learning journey, analyze their GitHub profile, generate personalized roadmaps, receive AI-powered career guidance, prepare for interviews, and visualize their skill growth.

---

## 🎯 Problem Statement

Students often struggle with:

* Choosing the right learning path
* Tracking DSA progress effectively
* Understanding their GitHub profile strengths
* Preparing for technical interviews
* Building a structured roadmap toward their goals
* Receiving personalized career guidance

CodeMentor AI solves these challenges through intelligent recommendations, progress tracking, and AI-powered mentorship.
